# shireen-sharma

Personal portfolio website for Shireen Sharma.

## Live Site

Once deployed, visit: `https://<your-github-username>.github.io/shireen-sharma/`

## Deploying to GitHub Pages

This is a **single-file static site** — everything (HTML, CSS, JS, and all images) is embedded in `index.html`. No build step or dependencies needed.

### Steps to publish

1. **Create a GitHub account** at https://github.com if you don't have one.

2. **Create a new repository** named `shireen-sharma` (or any name you like).
   - Go to https://github.com/new
   - Name: `shireen-sharma`
   - Leave it Public
   - Do NOT initialize with README (you already have one)
   - Click "Create repository"

3. **Upload your file** (easiest method — no terminal needed):
   - On the repo page, click **"uploading an existing file"**
   - Drag and drop `index.html` (rename shireen.html → index.html before uploading)
   - Also upload this `README.md`
   - Commit message: `Initial deploy`
   - Click **Commit changes**

4. **Enable GitHub Pages**:
   - Go to your repo → **Settings** → **Pages** (left sidebar)
   - Under "Source", select **Deploy from a branch**
   - Branch: `main` / Folder: `/ (root)`
   - Click **Save**

5. **Wait ~60 seconds**, then visit:
   `https://<your-github-username>.github.io/shireen-sharma/`

---

## Updating the site

To update any content:
1. Download `shireen.html` from Claude after making changes
2. Rename it to `index.html`
3. Go to your GitHub repo → click `index.html` → click the pencil icon (edit) → drag/upload the new file
4. Commit the change
5. The site updates automatically within ~30 seconds

## Custom domain (optional)

To use `shireensharma.com` or similar:
1. Buy a domain (Namecheap, Google Domains, etc.)
2. In your repo Settings → Pages → Custom domain, enter your domain
3. Follow GitHub's DNS instructions for your domain registrar
